file - load symbols from executable compiled with flag -g

layout next - enter tui mode
C-x C-a, C-x a, C-x A - exit tui mode
tui enable
tui disable
C-p, C-n, C-b, C-f - get previous/next command in tui mode

frame - prints where you are
where - prints stack trace
print - print variable in current debugging context
info variables - print all global and static variables
info locals - print local variables of current stack frame
info args - print arguments of the current stack frame


next - step over
step - step into
finish - continue until return
